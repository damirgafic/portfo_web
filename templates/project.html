02Box Office Data Analysis

Python Project

Credits

  • Data set api from boxofficemojo.com

Contriving Various Graphs From the Data Set

Using a database with data on Box Offices, I used the Seaborn library to help me visualize the data into graphs. In my foundings I found that revenue and budget have a strong correlation. I found that movies with a website are more likely to perform better at the box office compared to those that don't. I also found that "last" and "Man" are the most common words found in movie titles. There are also common words used to describe movies that will most likely make a movie more succesful in the box office. Overall there are many different variables that can factor in weather a movie will be succesful in the box office.
